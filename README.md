# 🎮 GTA VI Intelligence Dashboard

> A complete end-to-end Data Analytics project that transforms gaming data into actionable business insights using **Python, SQL, MySQL, and Power BI**.

![Dashboard Preview](assets/dashboard-preview.png)

---

# 📖 Overview

The **GTA VI Intelligence Dashboard** analyzes the commercial performance, global market demand, trailer engagement, and revenue projections for one of the most anticipated game launches in history.

The project follows a complete analytics workflow:

- 📥 Data Collection
- 🧹 Data Cleaning
- 🗄 SQL Analysis
- 📊 Interactive Power BI Dashboard
- 📈 Business Insights

This project demonstrates how publicly available data can be transformed into executive-level dashboards for decision-making.

---

# 🎯 Business Problem

Gaming companies rely on historical sales, customer interest, geographic demand, and marketing performance to estimate the commercial success of upcoming game launches.

This project answers questions such as:

- How did the GTA VI trailer perform compared to GTA V?
- Which countries generate the highest revenue?
- How has GTA V grown since launch?
- What is the projected revenue of GTA VI under different launch scenarios?
- How has global search interest evolved over time?
